# Real Estate Sales & Revenue Analysis Dashboard

The Real Estate Sales & Revenue Analysis Dashboard is an interactive Business Intelligence project developed to analyze property sales performance, customer behavior, revenue generation, profit trends, and geographical distribution of real estate transactions.

The dashboard transforms raw real estate data into meaningful insights through interactive visualizations, enabling users to monitor business performance, identify profitable opportunities, and support data-driven decision-making.

The project was developed using Power BI, Excel, CSV datasets, and data visualization techniques to provide a comprehensive overview of the real estate business.

## 🎯 Project Objectives

* Analyze property sales performance.
* Monitor revenue, profit, and expenses.
* Understand customer purchasing behavior.
* Track property distribution across locations.
* Visualize key business metrics through interactive dashboards.
* Support strategic business decision-making.
